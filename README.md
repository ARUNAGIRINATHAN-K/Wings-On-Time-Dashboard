**Wings on Time ✈️ — Flight Performance Dashboard**

Wings on Time is a professional-grade, web-based dashboard application that visualizes airline performance and delay analytics in real time. Powered by IBM Cognos and styled with Bootstrap 5, it provides key insights for airline operators, analysts, and decision-makers to enhance operational efficiency and customer satisfaction.

**🚀 Features**

Executive Summary: Key KPIs like average delay, on-time rate, and satisfaction

Interactive Dashboard (IBM Cognos embedded)

Analytics Tab: Charts & heatmaps for delay patterns, satisfaction analysis, and more

Reports Tab: A complete project report with insights and recommendations

Secure Login Page with modern UI

Responsive Design using Bootstrap 5

Navigation Bar with Home, Dashboard, Analytics, Reports, Settings

Live Airline Performance Leaderboard

Export options: PDF & Excel

**📊 Key Technologies**

HTML5, CSS3 (Bootstrap 5)

JavaScript (for interactivity)

IBM Cognos (for real-time dashboard integration)

FontAwesome Icons

GitHub Pages compatible
